# Air Quality
## 소개
OPEN API를 활용하여 서울 및 부산의 대기질 수치를 통합 제공합니다. 

## 기능
- 도시별 평균 대기질 수치
- 도시 구역별 상세 대기질 수치
- API 캐싱
